# PySpark-GastosMEF
En estre proyectos se realiza la canalización de datos que ingiere, transforma y analiza datos de gasto del gobierno peruano del Ministerio de Economía y Finanzas (MEF). El sistema extrae registros de gastos diarios de archivos CSV publicados en el portal de datos abiertos de Perú (datosabiertos.gob.pe), los transforma a un formato columnar optimizado utilizando Apache Spark y los almacena en un datamart para consultas analíticas.
